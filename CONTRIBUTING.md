# Contributing to VXLIB

## Status

This repository is a **read-only public mirror**. Community contributions are not accepted at this time.

Bug reports and feature requests are welcome via [TI E2E Community Forums](https://e2e.ti.com).

## Reporting Bugs

When filing a bug on E2E, include:

- **Device** — AM62Dx or AM275x
- **Toolchain** — `ti-cgt-c7000` version
- **Kernel** — which `VXLIB_*` function
- **Variant** — `OPT` (`_ci`) or `NATC` (`_cn`)
- **Steps to reproduce** — minimal code snippet
- **Expected vs actual output**
